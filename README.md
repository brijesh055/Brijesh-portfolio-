# Brijesh Portfolio | 3D Interactive Developer Portfolio

A modern, interactive 3D portfolio built with React, Three.js, and Tailwind CSS. Showcasing projects, skills, and experience as a Software Engineer.

![Portfolio Preview](./public/bky.png)

## Live Demo

🚀 **Live Site**: [https://brijesh055.github.io/Brijesh-portfolio-](https://brijesh055.github.io/Brijesh-portfolio-)

## Features

- **3D Particle Galaxy Hero** - Interactive Three.js background with rotating geometries
- **Glassmorphism UI** - Modern frosted glass design with smooth animations
- **Custom Animated Cursor** - Glowing cursor with hover effects
- **Responsive Design** - Mobile-first, works on all devices
- **Scroll Animations** - Framer Motion powered transitions
- **Dark Theme** - Space-inspired color scheme with electric blue accents

## Tech Stack

| Category | Technology |
|----------|------------|
| Framework | React 18 |
| Build Tool | Vite |
| 3D Rendering | Three.js |
| 3D Components | React Three Fiber + Drei |
| Animations | Framer Motion |
| Styling | Tailwind CSS |
| Icons | Lucide React |
| Deployment | GitHub Pages |

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
